# collab-docs
Proof-of-concept real-time collaboration tool using NodeJS and Socket.IO

![Project Screenshot](https://img.shields.io/badge/Status-Beta-blue?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Stack-Node.js%20%7C%20Socket.IO%20%7C%20Vanilla%20JS-success?style=for-the-badge)

## Overview

Collab docs focuses on the core mechanics of realtime sync

## Stack

- **Backend**: Node.js, Express
- **Realtime**: Socket.IO (WebSockets)
- **Frontend**: HTML5, CSS, JavaScript
- **Fonts**: Outfit (Google Fonts)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16.x or later recommended)

### Installation

1. Clone or download the project.
2. Navigate to the project directory:
   ```bash
   cd collab-docs
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

1. Start the server:
   ```bash
   node server.js
   ```
2. Open your browser to:
   ```
   http://localhost:3000
   ```
